## 👀 프로젝트 개요
-  📆 프로젝트 기간
   -  2023.12.11 ~ 2024.1.12 (5 week)
-  📌 기획 배경
   -  지역상권 분석의 중요성을 강조하며, 예비 창업자들에게 최적의 업종 선택을 위해 업종별 폐업률과 매출 정보 분석이 필요

-  👓 메인 타겟
